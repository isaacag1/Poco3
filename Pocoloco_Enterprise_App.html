📊 Google Sheets Connection Guide SETUP
1Upload spreadsheet to Google Drive — open drive.google.com → drag the .xlsx file in → right-click → Open with Google Sheets 2Bookmark the spreadsheet — you'll open it daily to enter sales 3Save this app — bookmark this page or save the HTML file alongside it 4Daily workflow: Open app → select items → confirm sale → Copy CSV → go to Google Sheets → 📝 Sales Entry tab → click first empty row in col A → Paste values only (Ctrl+Shift+V) 5Sync new items: Add rows to 📋 Scoresheet in Google Sheets → copy → paste into 🔄 Sync SKUs panel here
Paste tip: In Google Sheets always use Ctrl+Shift+V (Paste values only) — not Ctrl+V. This prevents formatting conflicts.
